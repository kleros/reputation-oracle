Add a run summary that is emitted as the last log line on every exit path (per D-05, D-06).

1. **Track timing and counters in index.ts `main()`:**

   At the very top of `main()`, before anything else:
   ```typescript
   const startTime = Date.now();
   let summary: RunSummary = { items: 0, valid: 0, actions: 0, txSent: 0, errors: 0, durationMs: 0 };
   ```

2. **Update counters as the bot progresses:**

   After fetching items: `summary.items = rawItems.length;`
   After validation: `summary.valid = validItems.length;`
   After computing actions: `summary.actions = actions.length;`

   For dry-run path: summary is emitted before return (items, valid, actions filled, txSent=0, errors=0).

   For execute path: the `executeActions` function currently does not return tx count. Instead, after `executeActions` completes without throwing, set `summary.txSent = actions.length;` (all actions succeeded since stop-on-first-failure).

3. **Emit summary on all exit paths:**

   Create a helper function at module level:
   ```typescript
   function emitSummary(summary: RunSummary, startTime: number): void {
     summary.durationMs = Date.now() - startTime;
     logger.info({ summary }, "Run complete");
   }
   ```

   Call `emitSummary(summary, startTime)` at:
   - End of dry-run path (before `return`)
   - End of no-actions path (before `return`)
   - After successful executeActions (before `return` or end of main)
   - In the `.catch()` handler (set `summary.errors = 1` first, then emit)

4. **Update the `.catch()` handler at the bottom of the file:**

   The catch handler needs access to `summary` and `startTime`. Restructure main() to use try/catch internally:

   ```typescript
   async function main(): Promise<void> {
     const startTime = Date.now();
     const summary: RunSummary = { items: 0, valid: 0, actions: 0, txSent: 0, errors: 0, durationMs: 0 };

     try {
       // ... existing logic with counter updates ...

       emitSummary(summary, startTime);
     } catch (error) {
       summary.errors = 1;
       emitSummary(summary, startTime);
       throw error; // re-throw so outer handler sets exit code
     }
   }
   ```

   The outer `.catch()` becomes:
   ```typescript
   main()
     .then(() => process.exit(0))
     .catch((error) => {
       logger.error({ err: error }, "Bot failed");
       process.exit(1);
     });
   ```

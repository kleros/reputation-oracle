Create `contracts/script/Verify.s.sol` as a Forge Script that verifies all 4 VER requirements. Follow the Deploy.s.sol pattern for Script structure and env var reading.

**Structure:**
1. Import Script, console from forge-std, IReputationRegistry, KlerosReputationRouter
2. Constants: `REPUTATION_REGISTRY = 0x8004B663056A597Dffe9eCcC1965A193B7388713`
3. `run()` function:
   a. Read `ROUTER_PROXY_ADDRESS` via `vm.envAddress("ROUTER_PROXY_ADDRESS")`
   b. Read `AGENT_IDS` via `vm.envString("AGENT_IDS")` — comma-separated list of agentIds to verify (per D-07, the agents the bot touched)
   c. Parse agentIds from the comma-separated string using a simple loop (split on `,`, parse each uint256)
   d. Create `address[] memory clients` array with just the router proxy address
   e. For each agentId, call and log:
      - `getSummary(agentId, clients, "", "")` — unfiltered (VER-01: count=1, value=95 for Scenario 1)
      - `getSummary(agentId, clients, "verified", "")` — tag1 filter (VER-04)
      - `getSummary(agentId, clients, "removed", "")` — tag1 filter (VER-04)
      - `router.feedbackType(agentId)` — Router state check
   f. Assert for Scenario 1 agents (per D-09): unfiltered count == 1, value == 95
   g. Assert tag filtering (per VER-04): "verified" filter returns count=1 for Scenario 1 agents, "removed" filter returns count=0
   h. Log a summary table to console

**Env vars:**
- `ROUTER_PROXY_ADDRESS` — deployed Router proxy address (required)
- `AGENT_IDS` — comma-separated agentIds to verify, e.g. "610,1142,1143,1440"
- `SEPOLIA_RPC_URL` — Sepolia RPC (used via --rpc-url flag, not read in script)

**Console output format (for D-08 report capture):**
```

  Router feedbackType: Positive                   [PASS]

... (repeat for each agent)

=== All assertions passed ===
```

**Important:** The script MUST use `require` (Solidity assert) for assertions so that `forge script` exits non-zero on failure. Use `require(count == 1, "...")` style.

**Note on agentId parsing:** Forge's `vm.envString` returns the full string. To parse comma-separated uint256 values, implement a simple `_parseAgentIds(string memory)` helper that:
- Iterates bytes of the string
- Splits on `,` (0x2c)
- Converts each segment to uint256 via `vm.parseUint(segment)`
- Returns `uint256[] memory`
